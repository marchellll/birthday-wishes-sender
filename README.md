# Surya Digital

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# copy the env
$ cp env.example .env

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
